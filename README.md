# Accel Radius — Landing Page

Halaman landing Astro + React untuk **Accel Radius Licensing**, dengan desain Swiss Design dan konten Bahasa Indonesia.

## Stack

- [Astro](https://astro.build/) — static site
- [React](https://react.dev/) — komponen interaktif
- [Tailwind CSS v4](https://tailwindcss.com/)
- [React Bits](https://reactbits.dev/) — animasi UI (SplitText, BlurText, FadeContent, dll.)
- [Bun](https://bun.sh/) — package manager

## Pengembangan

```bash
bun install
bun run dev
```

Buka http://localhost:4321

## Build

```bash
bun run build
bun run preview
```

## Branding

Aset branding (logo, ikon, favicon) disalin dari repositori utama `frontend/public/`.
